## 关键概念速记
| 概念 | 一句话解释 | 生活类比 |
|------|-----------|---------|
| Vibe Coding | 用自然语言描述需求，让AI帮你写代码 | 像跟建筑师描述你想要的房子，他帮你画图纸 |
| Agent Workflow | Agent拆解任务、分步执行的工作流 | 像项目经理把大任务拆成小步骤分配执行 |
| Tool Use | Agent调用外部工具（API、数据库等）完成任务 | 像人遇到不会的就去查Google、用计算器 |
| Agentic Commerce | Agent代替人进行商业交易和支付 | 像一个自动帮你比价、下单、付款的超级管家 |
| AI Security/Privacy | AI在Web3中如何保护数据隐私和安全 | 像给AI助手装上保险箱和监控摄像头 |
