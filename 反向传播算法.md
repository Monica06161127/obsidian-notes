# 反向传播算法
## 一句话解释
计算梯度的算法。
**利用微积分链式法则，通过从输出层向输入层反向计算误差对每个权重和偏置的敏感度（即梯度），从而精确指导神经网络如何调整参数以有效减小代价函数的数学机制。**

## 核心概念
 - 调整激活值的方式：
     - 调整权重，微调训练
     - 改变上一层的激活值
     - 改变偏置
- 降低代价：不只是每个参数变化，还有对应比例的变化大小
- Mini-batch：将庞大的训练数据集切分成的“小份样本集”。
- 反向应用链式法则：计算代价函数对之前的权重和偏置的敏感度
