# 🧠 AI × Web3 概念闪卡测验工具

> Week 1 可交互 AI 学习产物

## 解决什么问题？

Week 1 学了 20+ 个概念，容易遗忘。通过**主动回忆**（Active Recall）巩固记忆。

## 如何使用

```bash
python3 quiz.py
```

功能：
1. 随机测验（推荐）
2. 按类别浏览（AI / Web3）
3. 搜索概念
4. 查看所有概念

## 包含的概念（17 个）

**AI（7 个）**：LLM、Prompt、Context Window、Agent、Tool Use、Workflow、Human-in-the-Loop

**Web3（10 个）**：Account、Address、Wallet、Seed Phrase、Private Key、Signature、Transaction、Gas、Smart Contract、Testnet、Block Explorer、Write vs Read

## 输入输出示例

```
❓ LLM（大语言模型）是什么？
💡 提示：想想一个读过很多书但从没实践过的人

[按回车]

✅ 答案：LLM 是读过海量文本的'超级接话王'——不真正理解，但能猜出最合理的回答。
📝 例子：问 Claude '什么是比特币？'，基于训练数据拼出回答。
⚠️ 误区：会'一本正经地胡说八道'（hallucination）。
🎯 关键点：LLM 是模式匹配，不是真正理解
```

## AI 辅助说明

**AI 辅助**：概念解释、生活类比、代码框架
**人工修改**：逐条复核改写、加入真实经历、测试功能、设计交互逻辑

## 限制

1. 知识库静态，不会自动更新
2. 没有学习进度记录
3. 没有错题本功能
4. 只有命令行界面
