# Agentic Commerce（Agent自主交易）

## 一句话解释
让AI Agent代替人类进行商业交易——自动比价、下单、支付、结算，全程不需要人手动操作。

## 生活类比
> 想象你有一个超级管家：你说"帮我用最低价买一张下周三从北京到上海的高铁票"，管家自动去12306查、比价、选座、付款，最后把车票发给你。Agentic Commerce就是把这个场景搬到数字世界。

## 为什么需要Web3
| 中心化方案的问题 | Web3怎么解决 |
|----------------|-------------|
| Agent用你的账号付款，你敢吗？ | 链上签名授权，你可以设限额 |
| 交易记录在平台手里 | 链上收据，不可篡改 |
| Agent可能被操纵多付钱 | 智能合约锁定支付规则 |
| 跨平台支付很麻烦 | 加密货币天然跨境无摩擦 |

## 核心问题拆解（Week 2方法论）
共学营会要求你拆解这个框架：
- **谁发起任务**：用户 / Agent
- **谁执行**：AI Agent
- **谁付款**：用户的钱，Agent代付？
- **谁验证**：链上记录 / 第三方？
- **谁承担风险**：Agent出错了谁负责？
