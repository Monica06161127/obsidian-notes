# 以太坊入门

## 一句话解释
支持全球资金转账，还能通过运行名为**智能合约**的代码来构建各种**去中心化应用（DApps）** 和组织的灵活区块链平台。

- ## 和比特币的区别

|           | 比特币                                                       | 以太坊                                                                                                 |
| --------- | --------------------------------------------------------- | --------------------------------------------------------------------------------------------------- |
| 定位（最本质）   | 一种**去中心化的数字货币支付系统**。作为一个全球性的公共账本，记录资金的转移，旨在替代或补充传统银行系统。   | **可编程的区块链平台**，不仅可以处理资金转账，还能运行名为**智能合约**（Smart Contracts）的代码，人们可以在以太坊上构建各种去中心化应用（DApps），如游戏、金融工具或组织。 |
| 功能        | 记账                                                        | 灵活，能运行代码                                                                                            |
| 共识机制      | **工作量证明（Proof of Work, PoW）** 矿工挖矿                        | **权益证明（Proof of Stake, PoS）**                                                                       |
| 原生代币与发行规则 | 它的总量是有限的，上限被设定为 **2100 万枚**。它的产生通过“区块奖励”实现，且奖励大约每 4 年减半一次 | **以太币 (ETH)**：以太坊网络的原生货币，主要用于驱动网络运行。它不仅是价值载体，还被用作**支付 Gas 费（燃料费）**，即在以太坊上运行代码或发送交易所需的费用。            |
