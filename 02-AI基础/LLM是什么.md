# LLM 是什么

## 一句话解释
大语言模型（LLM）本质上是由数十亿个参数组成的“下一词预测机”，它通过对互联网海量文本进行有损压缩，获得了模仿人类语言和逻辑的能力。

## 它怎么工作
- Token：模型处理信息的最小单位
- 预测下一个词：定一段文本序列，模型会根据其内部参数计算出每一个潜在单词出现的概率，并选择最可能的一个作为输出。
- 温度（Temperature）：推理阶段的一个超参数，用于调节输出的随机性。温度越高，模型输出越多样、更有“创意”；温度越低，输出则越趋向于高概率的确定性结果。

## 和传统程序的区别
|      | 传统程序                            | LLM                                                |
| ---- | ------------------------------- | -------------------------------------------------- |
| 规则   | 由程序员通过 C 或 Python 等语言编写的显式逻辑和算法 | 蕴含在数十亿个经过优化调整的**权重参数**中，其内部运作逻辑具有“不可解释性”           |
| 输出   | 严格遵循预设指令的确定性结果。                 | 通过对概率分布进行**采样**生成的预测文本，本质上是一种“联想”或“幻觉”。            |
| 可预测性 | 高。像汽车一样，人们了解每一个零件的功能和组合方式。      | 较低。LLM 是**经验性制品**，其行为需要通过大量的测试评估才能掌握，甚至会产生无法预料的偏离。 |

## 生活类比
 **刚毕业的大学生：** 指令微调后的模型（Instruction-tuned LLM）就像一名**聪明但对你的具体任务一无所知的职场新人**。如果你能给出清晰、具体的指令（Prompt），他就能利用其广博的背景知识完成任务。
