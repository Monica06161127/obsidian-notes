# Vibe Coding

## 一句话解释
用自然语言（大白话）描述你想要的功能，让AI帮你写代码——你负责"想清楚要什么"，AI负责"怎么实现"。

## 生活类比
> 就像你跟一个很厉害的厨师说"我想吃一道酸甜口的鱼"，你不需要知道具体怎么颠勺、怎么调酱汁，厨师会帮你搞定。你只需要知道自己想吃什么。

## 它解决了什么问题
- **编程门槛**：不会写代码的人也能用AI"写出"程序
- **效率**：会写代码的人用AI可以快5-10倍
- **核心能力变了**：从"会写语法"变成"会描述需求"和"会判断AI写的对不对"

## 在共学营中的位置
Week 1 的核心实操之一——你需要用 Vibe Coding 完成 HermesAgent 的实践。

## 关键认知
- Vibe Coding ≠ 不用懂技术。你仍然需要知道**需求合不合理**、**代码跑起来对不对**
- 就像你用计算器，仍然需要知道该算加法还是乘法
- 目前最好的 Vibe Coding 工具：Cursor、Claude、GitHub Copilot
